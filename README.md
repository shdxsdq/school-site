# 学校官网原型

## 已完成
- 顶部信息栏 + 主导航
- 大图轮播（自动播放 + 圆点切换）
- 学校动态 / 党建工作 / 教学教研三栏资讯
- 校长寄语与教师风采模块
- 安全工作与页脚
- 移动端自适应（含折叠菜单）

## 预览方式
1. 直接双击 `index.html`
2. 或在当前目录运行：

```powershell
python -m http.server 8080
```

然后访问 `http://localhost:8080`。
